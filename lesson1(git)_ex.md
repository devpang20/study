## 로컬 저장소 -> 원격 저장소 git remote 버전은 과제
### 힌트    
- git remote 
```git
$ git add '별칭' [remote repository URL 주소]
(대게 별칭은 처음에 origin 로 쓴다.)
$ git remote add origin https://github.com/test.git

remote 상태보기
$ git remote show "별칭"
```

- git init
```git
저장소 초기화
$ git init
git master branch 생성
```

- git push origin master 
```git
$ git origin master
```
